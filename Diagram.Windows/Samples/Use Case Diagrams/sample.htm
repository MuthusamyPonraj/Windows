Use Case Diagram

These samples cover aspects such as programmatically defining a symbol, creating dynamic symbols that respond to user events, loading symbols from a symbol palette designed using the symbol designer application, symbol model synthesis and creating some commonly used diagrams.

  • Flow Diagram - This sample illustrates how to create a flow diagram programmatically using the symbols designed with the Essential Diagram Symbol Designer utility.

  • Swim Lane Diagram - This sample illustrates how to create a swim-lane process diagram using the Essential Diagram.

  • Fishbone Diagram - This sample illustrates how to create an Ishikawa, or fishbone diagram using the Essential Diagram.

  • Network Diagram - This sample illustrates the use of the layers to create a diagram with distinct zones using the Essential Diagram.

  • Circuit Diagram - This sample illustrates how to design a circuit diagram using the Essential Diagram control.

  • FloorPlan Diagram - This sample illustrates how to create a FloorPlan diagram using the Essential Diagram control.

  • Database Diagram - This sample illustrates the creation of the database diagram with the given data source from a database using the Essential Diagram control.

  • Class Diagram - This sample illustrates the creation of the class diagram with the given class definition using the Essential Diagram control.

  • Sequence Diagram - This sample illustrates how to create a sequence diagram with the symbols designed by the Essential Diagram Symbol Designer utility.

  • Seat Reservation Diagram - This sample illustrates the flexibility of the diagram nodes by simulating a seat reservation system.

  • Weather Monitoring Diagram - This sample illustrates how to show the weather in USA, using the weather forecast web service defined by the NOAA in the Essential Diagram.
