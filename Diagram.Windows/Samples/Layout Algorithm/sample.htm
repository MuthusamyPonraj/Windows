Layout Algorithm

The samples in this section cover all aspects of layout management: from setting up a simple layout manager, to using the highly sophisticated pre-built layout managers that ship with Essential Diagram.

  • Directed Trees Layout - This sample demonstrates the DirectedTreeLayoutManager layout management class.

  • Hierarchic Trees Layout - This sample illustrates the HierarchicLayoutManager layout management class.

  • OrgChart Layout - This sample illustrates the OrgChartLayoutManager layout management class.

  • Radial Layout - This sample demonstrates the RadialTreeLayoutManager layout management class.

  • Symmetric Layout - This sample illustrates the SymmetricLayoutManager layout management class.

  • Table Layout - This sample demonstrates the TableLayoutManager layout management class.

  • Sub Graph Layout - This sample illustrates preferred layout positioning using the SubGraphLayoutManager layout management class.
