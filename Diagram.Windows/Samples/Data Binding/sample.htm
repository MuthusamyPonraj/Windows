Data Binding

This section cover all aspects of binding of data through XML, database and business object with Essential Diagram.

  • XML Binding - This sample illustrates how to generate/bind the diagram based on the given XML data source.

  • Database Binding - This sample illustrates how a DataTable [DB] can be mapped using Essential Diagram with the HierarchicalLayoutManager.

  • Diagram Data Binding - This sample illustrates how the database tables are mapped onto the Diagram control.

  • Business Object Binding - This sample illustrates the binding of diagram to the business object data collection with the manual layout.
