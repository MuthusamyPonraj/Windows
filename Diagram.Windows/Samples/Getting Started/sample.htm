Getting Started

The samples under this section illustrate the basic concepts and features of Essential Diagram, such as creating a diagram control, using drawing tools, serialization, printing, dragging and dropping, etc.

  • Nodes - This sample illustrates the various types of nodes which are available in Essential Diagram.

  • Connectors - This sample illustrates the various types of connectors which are available in Essential Diagram.

  • Lables - This sample illustrates how to create labels and add them to a node in Essential Diagram.

  • Ports - This sample illustrates creating and adding new ports to a symbol.

  • Layers - This sample demonstrates the use of layers to create a diagram with distinct zones.

  • Grid - This sample demonstrates the support for adding grid of evenly spaced points that provides a visual guide to the user.

  • Overview - The Overview sample demonstrates the "Zoom and Pan Overview Window" in Essential Diagram.

  • Multiple Views - This sample highlights the modularity and power of the Model View Controller (MVC) architecture used by Essential Diagram.

  • Custom Symbol - This sample illustrates how to customize the behavior of a symbol designed with the diagram symbol designer by subclassifying it at the application level.

  • Custom Selection Handle - This sample illustrates built-in support for customizing the appearance of selection handles.

  • Custom Command - This sample illustrates how to perform undo-redo operations in diagram applications.

  • Scripting - This sample illustrates how to add scriptable interactivity to diagrams.

  • Expander - This sample illustrates expandable and collapsible nodes functionality in Diagram.

  • Drag and Drop - This sample illustrates how to implement drag-drop support in a diagram application.

  • Simple Drill Down - This sample illustrates how to create a drill down diagram so users can follow a path through different levels.

  • Diagram Events - This sample illustrates the various Diagram.EventSink events supported by Essential Diagram.

  • Model Events - This sample illustrates the various Diagram.Model.EventSink events supported by Essential Diagram.

  • Save To Database - This sample illustrates the process of saving a diagram as a binary in the database.

  • Serialization - This sample illustrates the approach involved in creating a simple diagram and saving it to disk as an Essential Diagram document (*.edd file).

  • Localization Demo - This sample illustrates the localization of various diagram dialogs such as the Header-Footer dialog, Page Borders dialog, Print Setup dialog, and Layout dialog.

  • Controls Galore - This sample illustrates the new ControlNode class that enables embedding Windows Forms controls in a diagram.

  • Grouping and Ordering - This sample illustrates the grouping and ordering of node in Essential Diagram.

  • Line Routing and Bridging - This sample illustrates the enabling and disabling of LineBridging and LineRouting in Essential Diagram.

  • Zooming and Panning - This sample illustrates the zooming and panning support in Essential Diagram.
