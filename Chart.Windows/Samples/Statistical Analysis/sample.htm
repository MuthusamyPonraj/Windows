Statistical Analysis

Statistical formulas help users analyze information and create more meaningful data.

Statistical formulas are implemented using the Statistics class.  They can be organized into the following general groups: statistical tests, basic statistical functions, and utility functions.

  • Statistical Tests

    • ANOVA test, f-test, z-test formulas

    • T-test, basic statistical formulas and functions

  • Basic Statistical Functions

    • Mean, median, standard deviation, variance, correlation coefficient, and covariance

  • Utility functions

    • Normal distribution, t-cumulative distribution, and f-cumulative distribution
