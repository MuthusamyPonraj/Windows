Data Binding

Essential Chart provides a series-model implementation that works directly on top of grouped data.

  • Axis Label Binding- This sample demonstrates loading and displaying chart data from a database. It also illustrates how to bind axis labels in pie, funnel, and pyramid charts.

  • Chart Grouping-This sample demonstrates plotting filtered data using the grouping engine in Essential Chart. Many features of Essential Grouping have been implemented to filter out chart data.

  • Chart with Huge Data Source-This sample illustrates rendering 100,000 data points on a chart.

  • Custom Collections-This sample illustrates custom collections binding with the Essential Chart.

  • Filtered Data-This sample shows how to improve the performance of a chart when loading a large number of data points using IChartSeriesModel.

  • Huge Data in Multi-Chart-This sample shows how selections in a high-level view can be expanded to a low-level view when they are bound to large data.

  • LINQ Data Source- This sample illustrates the binding of a chart control to a custom data source, which is generated using a language integrated query.
