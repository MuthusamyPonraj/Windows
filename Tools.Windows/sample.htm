Syncfusion Essential Tools
Essential Tools is a great collection of UI components that help in rapid application development. It can be used to create robust and flexible application interfaces like .NET-style docking, .NET-style tabbed-MDI interfaces, fully customizable menus, and Office 2007-like UIs. Also included in the toolset are tabs, task menus, Outlook-like group bars, a range of text editors, and much more.
