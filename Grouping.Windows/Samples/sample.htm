Grouping Windows Samples

This section contains the samples for Essential Grouping. It walks you through Essential Grouping's important features, functionalities, and implementation process.

The following is a list of samples with a brief description of each.

  • Custom Summary - Illustrates the functionality of custom summaries and custom counters

  • Foreign-Key Reference - Shows the foreign-key-reference relation

  • Grouping Tutorial - Shows you how to access and use the grouping features

  • Grouping with Data Grid Demo - Illustrates a Windows Forms data grid and describes its functionality

  • List-Item Reference - Shows the list-item reference relation

  • Manual Relations - Demonstrates how to manually specify master-detail relations between different tables

  • Random Test Demo - Illustrates the use of System.Random, which generates values

  • Strong-Typed Collection - Features a custom collection bound to a hierarchical grid
