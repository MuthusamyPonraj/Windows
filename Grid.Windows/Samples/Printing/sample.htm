Print

The following samples illustrate the printing features of Essential Grid.

Print Grid Demo - This sample illustrates printing a grid on a single page.

Print to Fit Demo - This sample illustrates printing a grid on a single page and adding a print preview dialog box. This sample also enables you to change grid content by increasing or decreasing the number of rows and columns.

Print Page Layout Demo - This sample illustrates the layout of each page displayed in a grid.

Multi-GridPrinting - This sample illustrates the multiple grid printing .
