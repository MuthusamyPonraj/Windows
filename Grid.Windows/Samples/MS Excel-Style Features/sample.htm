MS Excel-Style Features

Essential Grid supports MS-Office features like dynamic splitters and undo-redo.

The different samples that illustrate the Excel-Style features of Essential Grid are listed below:

Mark Header Demo- This sample illustrates higlighting row and column behavior available in Essential Grid.

Workbook Demo- This sample illustrates the display of grids in a tabbed workbook format similar to Excel.

Freeze Pane Demo- This sample illustrates the implementation of the Freeze Pane feature in Essential Grid.

Selection Marker Demo- This sample illustrates Excel-Style selection behavior and Excel Marker selection behavior available in Essential Grid.

Hide Rows and Columns Demo- This sample demonstrates to display the Grid with hidden rows and columns
