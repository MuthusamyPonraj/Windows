Drag-and-Drop Support Demo

Essential Grid offers support functionality similar to OLE drag-and-drop support. You can drag a column header to a new position in a similar manner to how fields in Microsoft Outlook are dragged without selecting columns before dragging. It also allows you to select a range in one grid and drag it to another grid or another control.

The different demo samples that illustrate the drag-and-drop support features of Essential Grid are listed below:

Drag Column Header Demo- Demonstrates the drag column header’s mouse controller

OLE Drag-and-Drop Demo- Lets you select a range in one grid and drag it to another grid or a rich edit box.
