Grouping

The Grid table can be grouped with its column, where it will appear as a tree view.

The different samples included in this feature are listed below:

Grouping Demo - This sample illustrates how to group data at design time.

Multicolumn Grouping Demo - This sample lets you group data by multiple columns.

Product Ratings Demo - This sample shows a product table where each product has a rating from multiple customers.

Grouping Events Demo - This sample demonstrates the creation of an event log that will show the order of various grouping events that are fired.
