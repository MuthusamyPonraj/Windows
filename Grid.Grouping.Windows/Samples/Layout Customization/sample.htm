Layout Customization

The different samples listed below illustrate the Grouping Grid layout.

Group Customization Demo - This samples demonstrates how to customize the layout of groups.

Stacked Headers Demo - This sample shows the implementation of a Microsoft Outlook 2007-like interface using Essential Grid and other components from Essential Studio.

Field Chooser Demo - This sample demonstrates how to use a plug-in utility called Field Chooser, which will help you customize the view of a grid  based on column visibility.

Record Customization Demo - This sample demonstrates multiple rows per record and image cells.

Frozen Column Demo - This sample demonstrates how to freeze columns, in a Grid Grouping Control, that are bound to a hierarchical data set.

Resizable Rows Demo - This sample demonstrates the features of resizing individual rows in a grid.

Find-Replace Demo - This sample comes with added support for performing search-replace operations with the Grid Grouping Control.

Cut-Copy-Paste Demo - This sample allows you to perform cut, copy, and paste operations with a hierarchical Grid Grouping Control.
