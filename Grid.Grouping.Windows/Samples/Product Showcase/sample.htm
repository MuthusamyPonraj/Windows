Product Showcase

The samples provided in this section demonstrate the optimal use of Essential Grid Grouping in real-time applications.

Portfolio Grid Demo - The sample simulates a portfolio manager application using Essential Grid.

LINQ Group-by Demo - This sample demonstrates the grouping engine support for IEnumerable as a data source and a nested collection.

LINQ To SQL Demo - This sample demonstrates the implementation of the LINQ to SQL feature of the 3.5 .NET framework.

Outlook Demo - This sample shows the implementation of a Microsoft Outlook like interface using Essential Grid and other components from Essential Studio.

GridVsChart Demo - This sample provided in this section demonstrate to the bind a chart based on the selection/actual value in grid.
