Exporting

The Grid Grouping Control supports the conversion of content to XLS, PDF, and Word.

The different samples included in this feature are listed below:

 Excel Export Demo - This sample demonstrates the conversion of content in a Grid Grouping control to an XLS file.

PDF Export Demo - This sample demonstrates the conversion of content in a Grid Grouping control to a PDF file.

Word Converter Demo - This sample demonstrates the conversion of the content in a Grid Grouping control to a Word file.
