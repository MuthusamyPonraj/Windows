Syncfusion Essential Grid Grouping Control
The Grid Grouping Control is a column-row-oriented grid that is used to bind to a data source. Essential Grouping allows you to easily group, sort, filter, and summarize your data. It can display true nested grids with hierarchical data, and it can also display multiple, unrelated tables in one grid.
