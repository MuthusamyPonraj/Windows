Data Management

Excel can manage and manipulate a large range of different kinds of data.

This section contains samples that deal with various types of data and their manipulation, using Essential XlsIO.

Here is a list of samples with descriptions of each.

  • Range Manipulation - This sample demonstrates how a range of cells can be manipulated using XlsIO. This includes named ranges and range copy with and without formatting.  It also details how to delete a range and apply styles to a range of cells.

  • Data Sorting - This sample demostrates the data sorting with various sorting algorithm.

  • Formulas - This sample demonstrates various formulas supported by XlsIO and how those formulas can be read from the existing template

  • Compute All Formulas - This sample demonstrates how formulas can be read from a XlsIO-created document without refreshing it manually.

  • Compute Specific Formula - This sample demonstrates how XlsIO can be integrated with Essential Calculate to compute the value of a formula entered using XlsIO.

  • Data Validation - This sample demonstrates how simple data validation can be performed with only a few lines of code.

  • Interactive Features - This sample demonstrates various interactive features using XlsIO. It explains hyperlink types and comments that can be inserted.

  • Form Controls - This sample demonstrates how to insert form controls using XlsIO.

  • Performance - This sample demonstrates how to increase the performance XlsIO with less time and memory consumption.

  • OleObject - This sample demonstrates how to insert oleobjects using XlsIO.
