Business Intelligence

This category contains samples that deal with various manipulations that can performed with  XLSX file formats of Business Intelligence, using Essential XlsIO.

Here is the list of samples with short descriptions of each.

  • Pivot Table - This sample demonstrates how to create a pivot table with rich formatting.

  • Pivot Chart - This sample demonstrates how to create a pivot Chart using XlsIO.

  • Tables - This sample demonstrates how to create Tables using XlsIO.
