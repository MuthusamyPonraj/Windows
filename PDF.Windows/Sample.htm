Syncfusion Essential PDF
Essential PDF is a .NET library capable of producing Adobe PDF files. It features a full-fledged object model for easily creating PDF files from any .NET language. It does not use any external libraries, and it is built from scratch in C#. It can be used on the server side (ASP.NET or any other environment) or with Windows Forms applications. Essential PDF supports many features for a PDF document: drawing text, images, shapes, etc. All can easily be done with Essential PDF.
